# Price3D

A new project for 3D pricing analysis/visualization.

## Project Structure

- `rexx/`: REXX Execs and logic.
- `sql/`: SQL statements for SPUFI and DB2 interactions.
- `panels/`: ISPF Panel definitions.
- `jcl/`: (Optional) Deployment and execution JCL.
